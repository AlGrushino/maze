Создание идеального лабиринта с помощью алгоритма Эллера и поиск пути с помощью обхода в ширину(BFS), которые реализованы самостоятельно, также реализован графический интерфейс с  помощью библиотеки fyne, код покрыт документацией и тестами.

How to run:

git clone

cd maze/src

make

cd build

./maze

enjoy


How to test:

git clone

cd maze/src

make tests
